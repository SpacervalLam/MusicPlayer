# 音乐播放器 (Music Player)

一个基于 Qt/QML 和 C++ 开发的现代化音乐播放器。

## 应用截图

![应用截图](docs/image.png)

## ✨ 主要特性

### 🎵 核心播放功能
- 支持多种音频格式播放（基于 Qt Multimedia）
- 完整的播放控制：播放/暂停、上一首、下一首、进度控制
- 多种播放模式：顺序播放、单曲循环、列表循环、随机播放
- 音量控制和静音功能

### 🎨 界面设计
- **无边框窗口设计**：现代化的全屏界面体验
- **智能收纳功能**：支持窗口自动隐藏和边缘唤出
- **动态背景**：支持静态图片和 GIF 动图背景，可自定义多张背景切换
- **响应式布局**：自适应不同屏幕尺寸

### 🎼 音频可视化
- **实时频谱分析**：60频段音频频谱显示
- **音频电平监测**：实时音频强度指示
- **对称频谱显示**：左右镜像的频谱动画效果
- 基于 KissFFT 的高性能 FFT 处理

### 📱 播放列表管理
- 文件夹导入功能
- 播放列表显示和管理
- 当前播放状态指示

### 🎤 歌词显示 (to do)
- 支持歌词文件解析和显示
- 当前歌词和下一句歌词预览
- 时间同步的歌词滚动

### ⚙️ 系统集成
- **置顶窗口**：始终保持在最上层
- **全局鼠标检测**：智能的边缘检测和自动隐藏
- **设置持久化**：自动保存和恢复用户设置
- **Windows 集成**：原生 Windows 应用程序，无控制台窗口

## 🛠️ 技术栈

- **前端框架**: Qt 6.2+ (QML/Qt Quick)
- **后端语言**: C++17
- **音频处理**: Qt Multimedia + KissFFT
- **构建系统**: CMake 3.16+
- **UI 组件**: Qt Quick Controls 2

## 📋 系统要求

- **操作系统**: Windows 10/11
- **运行时**: Qt 6.2 或更高版本运行时库
- **内存**: 最少 512MB 可用内存
- **存储**: 约 120MB 安装空间

## 🚀 安装方式

### 方式一：下载安装包
1. 下载最新的 `MusicPlayerInstaller.exe`
2. 运行安装程序，按提示完成安装
3. 从开始菜单或桌面快捷方式启动应用

### 方式二：从源码构建
1. **克隆仓库**
   ```bash
   git clone <repository-url>
   cd player
   ```

2. **安装依赖**
   - 安装 Qt 6.2 或更高版本
   - 安装 CMake 3.16 或更高版本
   - 安装 C++17 兼容的编译器 (Visual Studio 2019+ 或 MinGW)

3. **构建项目**
   ```bash
   mkdir build
   cd build
   cmake ..
   cmake --build . --config Release
   ```

4. **运行应用**
   ```bash
   # Windows
   ./Release/app.exe
   ```
